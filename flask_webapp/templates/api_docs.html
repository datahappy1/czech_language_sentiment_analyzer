API Documentation

POST endpoint URL: http://czester.herokuapp.com/api/v1/prediction/

The API currently allows only POST requests with a singleton input text value provided in the Input_Text parameter.

  • Example CURL POST request:
    • curl -X POST -F Input_Text="your text for analysis" http://czester.herokuapp.com/api/v1/prediction/
  • Example CURL POST request success response:
    • {"mimetype":"application/json","sentiment_result":{"overall_sentiment":{"probability":0.72,"sentiment":"negative"}},"status":200}
  • Example CURL POST request error response for insufficient input text - missing at least 3 words:
    • {"error":"Sorry, need to submit at least 3 non stop-words","mimetype":"application/json","status":400}
  • Example CURL POST request error response for insufficient input text - missing at least 1 word with 3 and more characters:
    • {"error":"Sorry, need to submit at least 1 word with 3 and more characters","mimetype":"application/json","status":400}
  • Example CURL POST request error response for insufficient input text - Czech language not recognized:
    • {"error":"Sorry, need to submit text written in Czech","mimetype":"application/json","status":400}
  • Example CURL POST request error response for not found:
    • {"error":"404 Not Found: The requested URL was not found on the server. If you entered the URL manually please check your spelling and try again.","mimetype":"application/json","status":404}
  • Example CURL POST request error response for a bad request:
    • {"error":"405 Method Not Allowed: The method is not allowed for the requested URL.","mimetype":"application/json","status":405}
